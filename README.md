## Installation

To install this project:

1. Go to Repo and get ssh key
   
       git@github.com:RomanScandariato/Portfolio_Website.git

3. Navigate to where you want to clone the repo

        cd -

4. Clone the Repository

       git clone (ssh key)

6. Open Project in VSCode 

       code . -a
